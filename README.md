# Slider-Crank Mechanism Kinematic Analysis ⚙️

This repository contains the design, code, and analysis data for the Slider-Crank mechanism project developed as part of the MEE 203 Dynamics course.

## 📂 Repository Contents
* **Dynamic Term Project Code:** Arduino source codes for motor control and data acquisition.
* **Analysis Data (All Datas.xlsm):** Comparison data between theoretical and experimental results.
* **Mechanism Photos:** Visuals of the prototype and setup.
* **Reports & Docs:** Project description and final technical report.

## 🚀 Key Features
* **Hardware:** Arduino Nano, Rotary Encoder, MPU6050 Accelerometer.
* **Methodology:** Sensor fusion applied to correct MEMS sensor drift.
* **Results:** High accuracy position tracking with <2% error rate compared to theoretical models.
